# Lab 02: Star Topology LAN

## Objective
Build a star topology LAN using one switch and multiple PCs, and verify connectivity.

## Topology
![Topology](topology.png)

## IP Addressing

| Device | IP Address |
|------|------------|
| PC1  | 192.168.1.10 |
| PC2  | 192.168.1.11 |
| PC3  | 192.168.1.12 |
| PC4  | 192.168.1.13 |

## Configuration Steps
1. Added one switch and four PCs
2. Connected PCs using straight-through cables
3. Assigned IP addresses manually

## Verification
- Ping successful between all PCs

## Lessons Learned
- Star topology simplifies troubleshooting
- Centralized switching improves performance
