# Lab 05: DHCP on a Router

## Objective
Configure a router to automatically assign IP addresses to hosts using DHCP.

## Topology
![Topology](topology.png)

## Network Details
- Network: 192.168.10.0/24
- Default Gateway: 192.168.10.1

## DHCP Configuration
- Router configured as DHCP server
- Excluded IP range to avoid conflicts
- DNS server assigned via DHCP

## Verification
- PCs received IP addresses automatically
- Successful ping to the default gateway

## Lessons Learned
- DHCP reduces manual configuration errors
- Routers can centrally manage IP addressing
- Proper exclusions are critical in DHCP design
